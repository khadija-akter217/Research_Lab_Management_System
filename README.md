The Research Hub is a digital platform that facilitates research management and collaboration. It enables users to publish, organize, and access research work while also supporting project tracking, collaboration, and communication. The system is designed with role-based access so that different users can interact with the platform according to their responsibilities. Researchers can upload publications, can submit projects proposal, and admin/Lab leader can monitor overall activities. The project aims to build a secure, user-friendly, and scalable platform that fosters innovation, academic progress, and collaborative research.There are two types of users:
1.Lab Leader/Admin.
2.Researcher.

User: Admin

The Admin Panel is designed to give full control over the entire system, allowing them to manage researcher profile. The admin acts as the central authority who ensures smooth functioning by monitoring activities, approving requests, and maintaining data integrity.

•	Admin Login System: A secure authentication system that ensures only authorized admin can    access.

•	Dashboard: A user-friendly dashboard where the admin can see the number of total Members, Ongoing Projects, Pending Fund Requests, Booked Equipment.

•	View Member: Admin can see the list of all registered user details like username, email and role. keep track of active users and monitoring activity.

•	Manage Projects: Admin can add new user accounts by entering details such as name, username, and email. Update and delete user from project if any changes are required.

•	Post Announcements: Admin can create new announcements with details for giving any notice to the user.

•	Equipment Management: Admin can maintain the Equipment’s if their need or end of any equipment’s also can see the booked equipment’s. 

•	Grant Fund: Admin can see the fund request and approve or reject the request.

•	Generate Reports: Admin can generate the reports of member, project, fund and equipment’s history.

•	Settings: Admin can update their profile by changing username, email and password with validation.

•	Logout: After clicking logout admin will go to the login page again.

 
User: Researcher

•	Login System: A secure authentication system that ensures only authorized users can    access.

•	Dashboard: A user-friendly dashboard where the user can see the number of total publications, Ongoing Projects, pending project proposal and Pending Fund Requests.

•	Publication: Researcher can see the list of all publications and add new publication in their profile.

•	Assign Projects: User can see the new assigned project and their deadlines.

•	Project proposal: User can give their desire project proposal to the lab admin.

•	Notice: User can see new announcements with details uploaded by admin.

•	Fund Request: User can request funds for needing any equipment or any conferences.

•	Book Equipment: User can the booked equipment’s as per their need and must approve by lab leader.

•	Settings: User can update their profile by changing username, email and password with validation and manage their account.

•	Logout: After clicking logout user will go to the login page again.
